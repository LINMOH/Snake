# 贪吃蛇游戏 (Snake Game v1.3.2)

这是一个使用 Pygame 库编写的经典贪吃蛇游戏，具有现代化的功能和视觉效果。包括渐变背景、粒子效果、音效以及随机生成的地雷元素，使游戏更加富有挑战性。

## 游戏功能

- **蛇的控制**: 通过键盘方向键控制蛇的移动。
- **食物**: 吃掉食物时蛇会变长并增加分数。
- **地雷**: 游戏中包含小地雷和大地雷，碰到地雷会导致不同的游戏后果。
  - **小地雷**: 撞到后扣分并消失。
  - **大地雷**: 撞到后游戏结束。
- **粒子效果**: 吃掉食物时会生成美丽的粒子效果。
- **背景音乐与音效**: 游戏包含背景音乐和不同的音效（例如吃食物、碰到地雷、游戏结束音效）。
